WatchDog

class WatchDog

A watchdog timer for a thread. The watched thread will be that associated with the scheduler, typically this will be the platform's main thread scheduler a crash dump of the watched thread will be delivered to the observer. Do NOT subscribe to this on the thread you want to watch :-)

Parameters

scheduler - The scheduler for the thread of interest.

timeout - The watchdog timeout in timeUnits

timeUnit - The timeUnit
